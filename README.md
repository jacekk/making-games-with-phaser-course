# Making Games With Phaser 2.X | Course

### Description

My implementation of this course: https://udemy.com/making-games-with-phaser .

### Requirements

- node ~8.x

### NPM Tasks

- **dev** - runs development server
- **build** - builds project to `dist` folder
- **clean** - remove `dist` folder

### Setup

1. `$ npm install`
1. `$ npm run dev`
